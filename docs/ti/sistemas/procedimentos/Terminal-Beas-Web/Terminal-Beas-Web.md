# Terminal Beas Web

---

## Informações Gerais

Este procedimento tem como objetivo instruir sobre a realização do cadastro de colaboradores que utilizam o Terminal Web do sistema b.eas.

## 1. Cadastro de Colaborador 

Para iniciar, é necessário acessar a tela de colaboradores dentro do sistema b.eas.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/Menucolaboradores.png)

Após acessar a tela de colaboradores, clique com o botão direito do mouse e selecione a opção **Adicionar**.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/botaoAdicionar.png)

A tela de cadastro de colaboradores será aberta. Nela, preencha as informações sublinhadas na imagem abaixo. 

> **Atenção:** Para o preenchimento do campo "Usuário", é necessário realizar previamente o cadastro do colaborador no sistema SAP.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/telaColaboradores.png)

Acesse a aba **Login** (demarcada na imagem) para criar o usuário e a senha do colaborador. Adicione o nome de usuário (utilizando o padrão `nome.ultimonome`) e a senha. 

> **Adendo:** A senha do Terminal b.eas não permite alterações pelo próprio usuário, sendo fixa desde o momento da criação.

O campo **Válido até** retrata a validade de acesso do usuário. Adicione a data limite do sistema, conforme mostrado no *print* a seguir:

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/loginTelaColaboradores.png)

## 2. Permissões e Acesso a Formulários via Tablet

Com o usuário devidamente cadastrado, acesse o menu indicado no caminho da imagem abaixo para incluir o colaborador nos *templates*.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/templatesOrdensCq.png)

No sistema, após a abertura do menu, aplique o filtro clicando no ícone destacado para retornar os formulários a serem cadastrados.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/filtroTemplates.png)

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/templatesAdministrador.png)

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/ordensCQ.png)

Selecione o formulário no qual o colaborador será adicionado e, com um clique duplo, acesse as configurações do formulário.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/ordensCQ.png)

Em seguida, clique no ícone de **três pontos** (...) presente na linha demarcada na imagem a seguir e adicione o colaborador desejado.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/aplicaçãoForms.png)

Clique em **Atualizar** e depois em **OK** para confirmar a adição do colaborador ao formulário.

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/atulizarFiltroColaborador.png)

![](/ti/sistemas/procedimentos/Terminal-Beas-Web/atualizarForms.png)
