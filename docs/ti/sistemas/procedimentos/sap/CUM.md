# Procedimento para Cadastro de Conversões de UM

---

## Objetivo

Este documento visa esclarecer o processo de conversão de Unidade de Medida (UM) dentro do SAP, de forma simples e direta, para auxiliar a equipe no dia a dia.

## Regra de Conversão

O **Grupo de UM** cadastrado no sistema é a base central das conversões. Ele é aplicado automaticamente em ambas as pontas da operação:

* **Documentos de Compra**
* **Documentos de Venda**
