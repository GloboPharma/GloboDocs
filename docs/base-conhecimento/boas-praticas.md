# Boas Práticas de Documentação

Este documento estabelece os padrões para a criação de novos manuais no GloboDocs.

## 1. Escrita
- Use linguagem clara e direta.
- Evite parágrafos muito longos.

## 2. Formatação
- Utilize títulos hierárquicos (# para título, ## para subtítulo).
- Destaque comandos de terminal com blocos de código.

## 3. Imagens
- Salve todas as imagens na pasta `/assets`.
